# Личный финансовый менеджер

Это приложение для управления личными финансами. Оно позволяет отслеживать доходы и расходы, устанавливать лимиты для категорий и следить за балансом. Также предусмотрена возможность экспорта данных о кошельке в текстовый файл.

## Как пользоваться приложением

После запуска приложения будет предложено пройти авторизацию или зарегистрироваться.

### 1. **Регистрация**

- При первом запуске приложения, если пользователя с таким именем нет, будет предложено создать нового пользователя. Для этого нужно ввести имя пользователя и пароль.

### 2. **Вход**

- Если пользователь уже зарегистрирован, достаточно ввести имя пользователя и пароль для входа в систему.

### 3. **Меню**

После входа в систему доступны следующие действия:

1. **Добавить доход**: Добавление нового дохода. Необходимо ввести категорию (например, "Зарплата" или "Пополнение") и сумму.

2. **Добавить расход**: Добавление расхода. Нужно выбрать категорию (если она еще не существует, она создается автоматически с пустым лимитом) и сумму.

3. **Установить лимит для категории**: Установка лимита для выбранной категории расходов.

4. **Просмотреть информацию о кошельке**: Отображение информации о кошельке, включая UUID, баланс, общий доход и расходы, а также доходы и расходы по категориям.

5. **Выход**: Завершение работы с приложением и сохранение всех данных.

## Поддерживаемые команды

1. **1** — Добавить доход
2. **2** — Добавить расход
3. **3** — Установить лимит для категории
4. **4** — Просмотреть информацию о кошельке
5. **5** — Выход

## Тестирование кода

### 1. **Запуск приложения**

Для запуска приложения необходимо:

- Скомпилировать код с помощью IDE (например, IntelliJ IDEA или Eclipse) или через командную строку.
- Запустить приложение, скомпилированное в виде `.jar` файла или через IDE.

### 2. **Тестирование функций**

Приложение поддерживает следующие действия:

- **Добавление дохода**: Ввести команду `1`, выбрать категорию и указать сумму.
- **Добавление расхода**: Ввести команду `2`, выбрать категорию и сумму.
- **Установка лимита**: Ввести команду `3`, выбрать категорию и установить лимит.
- **Просмотр информации о кошельке**: Ввести команду `4`, чтобы увидеть статистику о кошельке.
- **Выход**: Ввести команду `5` для выхода и сохранения данных.

После добавления дохода или расхода приложение автоматически выводит сообщение, если превышен лимит по какой-либо категории.

## Структура проекта

- **Store.java** — хранит данные пользователей и их кошельков.
- **Wallet.java** — управляет кошельком, транзакциями и выводом информации.
- **Transaction.java** — представляет транзакцию (доход/расход).
- **Category.java** — представляет категорию для доходов и расходов.
- **User.java** — представляет пользователя и его данные для авторизации.
